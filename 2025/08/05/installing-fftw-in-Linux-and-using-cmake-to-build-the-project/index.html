安装步骤

  1. 去fftw官网下载tar包。https://www.fftw.org/fftw-3.3.10.tar.gz。作者这里贴出3.3.10版本，且建议下载安装这个版本，因为3.3.8中缺少一个.cmake文件

中间有一行include ("${CMAKE_CURRENT_LIST_DIR}/FFTW3LibraryDepends.cmake")，d但可以发现此路径下缺少一个FFTW3LibraryDepends.cmake。官方的github也有人问了这个问题，详见https://github.com/FFTW/fftw3/issues/130。原因是这个文件只有使用cmake安装时才会出现，autotools并没有。我打算手动添加一下这个文件。
7. 还在刚才这个路径下。创建build目录并使用cmake编译文件
